タイトル 

Raspberry Pi5を用いたデスクワーク専用の勉強管理システム

動機

　勉強中に、離席する、居眠りをする、携帯をいじるなどの
 勉強をしてない時間を除いた、本当に勉強している時間のみを計測したい

目的

  ユーザーが「離席していないか」、「居眠りをしていないか」、「携帯を操作してないか」を
  USBカメラから取得した画像から、顔のランドマーク検出、物体検出を用いて判断し、
  実際に勉強していた時間を計測・記録し、勉強時間をグラフで可視化するシステムを開発する


・要求機能

    ・タイマーの場合は残り時間を、カウンターの場合は経過した時間をLCDディスプレイに表示
    
    ・USBカメラと、顔のランドマーク検出専用のモデルを用いて睡眠の判定に使う顔のランドマークの座標を検出
    
    ・検出した顔のランドマークの座標をもとにユーザの両目の瞼の開き具合を数値化し、数値が閾値を下回ったら寝ていると判定し、勉強時間の計測を止め、圧電ブザーを鳴らす
    
    ・目が開いている時のみ、スマートフォン検出用のTFLiteモデルを使いスマートフォンの検出を行う
    
    ・計測を終了させたいときは、カウンターの場合は、トグルスイッチを押すと終了し、タイマーの場合は、トグルスイッチを押すか、タイマーの残り時間が0になれば終了する
       
    ・勉強時間の計測が終了したら、日付と勉強時間をMySQLを使いデータベースに保存する
    
    ・7日間分の勉強時間のデータを対話形式のAI(Gemini)に渡し、データの分析を行わせる
    
    ・7日間分の勉強時間の棒グラフをpandasとmatplotlibで作成する。
    
    ・計測した勉強時間(テキスト形式)と、保存した棒グラフ(画像)、AIによる分析結果をemailで送る
    
    ・notionに今まで記録した全てのデータを表形式と棒グラフ形式で表示する (notionはメモやタスク管理、ドキュメント管理、などのさまざまなツールを1つにまとめたアプリケーション)
